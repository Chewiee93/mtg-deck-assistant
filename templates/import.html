Import Deck

How to import:
• Type in a deck name
• Choose deck format
• One card per line
• Format: "4 Lightning Bolt"
• Supports "4x" or "Card x4"
• Optional: specify set using "(SET)" e.g. "4 Lightning Bolt (M10)"
• Sideboard: use "Sideboard" then type cards in underneath
